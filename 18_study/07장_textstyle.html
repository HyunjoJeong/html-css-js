텍스트 속성들

텍스트 정렬: text-align 속성

줄 간격: line-height 속성(px, %, 2.1(=210%)), %나 배수는 글자 크기에 상대적.

텍스트의 줄을 관리하는: text-decoration 속성

- ex1: none

- ex2: underline

- ex3: overline

- ex4: line-through

- none을 이용하면 하이퍼링크의 밑줄도 지울 수 있다.

그림자효과: text-shadow 속성, 사용법: text-shadw: 오른쪽이동 아래쪽이동 (번짐정도) 색상 순으로 입력

uppercase, LOWERCASE, capitalize

글자 간격 조정: letter-spacing, word-spacing 속성

목록 스타일

불릿 모양을 변경하는 list-style-type 속성,
속성값: disc(채워진 원), circle(속이 빈 원), square(채워진 사각형), decimal, decimal-leading-zero(01, 02, ..), lower-roman, upper-roman, lower-alpha, upper-alpha, none

  • 불릿이 사각형인 list

불릿 대신 이미지를 사용하는 list-style-image 속성

  • 불릿 대신이미지를 활용한 list

리스트를 들여쓰는 list-style-position 속성

  • 들여쓰기(inside)
  • 기본값(outside)

각 속성을 한방에 쓸려면

  • 이미지로 바꾸고 들여쓰기
