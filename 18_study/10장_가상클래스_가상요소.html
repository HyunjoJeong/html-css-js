가상 클래스 선택자는 언제 사용하는 가?
- 사용자가 웹 요소를 클릭하거나 마우스 포인터를 올려놓는 등 특정 동작을 할 때 스타일이 바뀌도록 만들 때 쓴다.

1. 방문하지 않은 링크에 스타일을 적용하는 :link 가상 클래스 선택자와
2. 방문한 링크에 스타일을 적용하는 :visited 가상 클래스 선택자

방문했을까?(안 하면 파랑, 했으면 빨강)

3. 특정요소에 마우스 포인터를 올려놓으면 스타일을 적용하는 :hover 가상 클래스 선택자

4. 웹 요소를 활성화했을 때 스타일을 적용하는 :active 가상 클래스 선택자
- 즉, 링크나 이미지 등 웹 요소를 클릭했을 때 적용.

5. 웹요소에 초점이 맞춰져있을 때 스타일을 적용하는 :focus 가상 클래스 선택자
- 예를 들어 텍스트필드 안에 마우스 포인터를 올려놓거나, tab 키에 의해 선택되었을 때

구조 가상 클래스

그냥 문법만 쓰겠음.. 귀찮다
- :only-child = 부모 안에 자식 요소가 하나뿐일 때 '자식' 요소를 선택
- A:only-type-of = 부모 안에 A 요소가 하나 뿐일 때 'A' 요소를 선택
- :first-child = 부모 안에 있는 모든 요소 중 첫 번째 자식 요소 선택
- :last-child = ~~~~~~~~~~~~~~~~~ 중 마지막 자식 요소 선택
- A:first-of-type = 부모 안에 있는 A 요소 중 첫번째 요소 선택
- A:last-of-type = 부모 안에 있는 A 요소 중 마지막 요소 선택
- :nth-child(n) = 부모 안에 있는 요소 중 n번째 자식 요소 선택
ㄴ n자리에 an+b와 같이 수식을 사용할 수도 있으며, 단순히 홀수 짝수 일때는 odd 또는 even 사용 가능, (n은 1부터 시작)
- :nth-last-child(n) = 부모안에 있는 요소 중 끝에서 n번째 자식 요소 선택
ㄴ 마찬가지로 n 자리에 수식 및 odd, even 가능은 한데 굳이 싶지? ㅎ
- A:nth-of-type(n) = 부모 안에 있는 A 요소 중 n 번째 요소 선택
- A:nth-last-of-type(n) = 부모 안에 있는 A 요소 중 뒤에서 n 번째 요소 선택.
ㄴ 위에 두개도 당연히 n 자리에 수식 및 odd, even 가능.

가상 요소

우선, 총 4개가 있음. "::first-line, ::first-letter, ::before, ::after"
1. ::first-line, ::first-letter - 요소의 첫번째 줄이나 글자에 스타일 적용.
ㄴ 참고할 점은, ::first-letter는 첫번째 '줄'의 첫 글자에만 적용됨.
2. ::after, ::before - 내용 앞뒤에 콘텐츠(텍스트, 이미지)를 추가할 수 있음. 밑에 예시로 리스트를 작성해보았음.
