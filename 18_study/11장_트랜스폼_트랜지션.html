Transform(변형) & Transition(전이)

Transform과 변형함수

기본 문법 = transform: 함수

translate(이동), scale(확대, 축소), rotate(회전), skew(왜곡) 함수가 있음.

ex:
translate(10px, 5px, 3px)
scale(2, 1.5)
rotate(40deg): 시계방향 40도 회전
skew(40deg)
** 참고: perspective(~px) = 깊이감 (설정시, 회전할 때 원근감을 줄 수 있음)
ex: 예시

Transition(전이) 속성

트랜지션이란 웹 요소의 스타일 속성이 바뀌는 것을 말한다.
ex: 배경색, 도형의 테두리, 위치 등등

트랜지션을 실행하려면 아래 속성을 사용하는데,

  1. transition-property: all | none | (속성 이름)
    ㄴ 트랜지션의 대상 지정. 여러 속성 변형시 transition-property: 속성1, 속성2, ...
  2. transition-duration: (시간)
    ㄴ 2s 를 쓰면 2초 동안.
  3. transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n, n, n, n)
    ㄴ 트랜지션 효과의 속도 곡선을 지정한다. 마지막항목의 n 값은 0~1 사이 값만 쓸 수 있다. 자세한 건 구글신 참고.
  4. transition-delay: (시간)
    ㄴ 지연 시간.

참고로 transition 속성은 개체의 style에 작성하지, :hover 처럼 특정 상황의 style에 작성하지 않는다.
