반응형 웹

반응형 웹 디자인은 웹 요소를 화면 크기에 맞게 재배치하고 각 요소의 표시 방법만 바꾸어 사이트를 구현한다.

반응형 웹 디자인에서는 "viewport"를 지정하는데, 이는 화면에서 웹 문서 내용이 표시되는 실제 영역을 의미한다.
기본 문법: (meta name="viewport" content="속성1=값1, 속성2=..")
속성으로는 width, height, user-scalable, initial-scale이 있다.

viewport의 속성
종류설명가능한 값기본값
width뷰포트 너비device-width 또는 크기브라우저 기본값
height뷰포트 높이device-height 또는 크기브라우저 기본값
user-scalable확대/축소 가능 여부yes 또는 noyes
initial-scale초기 확대/축소 값1 ~ 101

뷰포트 단위

vw(viewport width) : 1vw는 뷰포트 너비의 1%
vh(viewport height) : 1vh는 뷰포트 높이의 1%
vmin(viewport minimum) : 1vmin은 뷰포트 너비와 높이 중 작은 값의 1%
vmax(viewport maximum) : 1vmax는 뷰포트 너비와 높이 중 큰 값의 1%

브라우저에서 디바이스 모드 활용하는 방법:
"여기"를 클릭, F12를 눌러 개발자 도구 창을 연 후 디바이스 모드 아이콘을 선택.

미디어 쿼리

media query란, 사이트에 접속하는 장치에 따라 특정한 CSS 스타일을 사용하는 방법을 의미한다.
미디어 쿼리를 이용하면 접속하는 장치의 화면 크기에 따라 레이아웃을 다르게 할 수 있다.

기본 문법: @media [only | not] 미디어 유형 [and 조건] * [and 조건] * ...
- only : 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 미디어쿼리를 무시하고 실행하지 않는다.
- not : not 다음에 지정하는 미디어 유형을 제외한다. ex) not TV - and : 조건을 여러 개 연결해서 추가할 수 있다.

미디어 유형의 종류
종류설명
all모든 미디어 유형에서 사용할 CSS
print인쇄 장치에서 사용할 CSS
screen컴퓨터(+스마트폰, 태블릿) 스크린에서 사용할 CSS
tv음성과 영상이 동시에 출력되는 TV에서 사용할 CSS
aural음성 합성 장치(화면을 읽어 소리로 출력하는)에서 사용할 CSS
braille점자 표시 장치에서 사용할 CSS
handheld패드처럼 손에 들고 다닐 장치를 위한 CSS
projection프로젝터를 위한 CSS
tty디스플레이 기능이 제한된(px를 못 쓰는) 장치에 맞는 CSS
embossed점자 프린터에서 사용할 CSS

미디어 쿼리의 조건

미디어 쿼리의 조건으로 웹 문서의 가로 너비나 세로 높이 속성을 사용할 수 있는데,
width, height, (min/max)-(width/height)가 그것이다.
* 참고로 height는 미디어의 종류에 따라 의미가 달라지니 조심해야 한다.
- screen이 아닌 미디어에서는 스크롤을 포함한 전체 문서를 height로 지정해야 하고,
print에서는 한 페이지 높이를 기준으로 한다.

단말기의 가로 너비, 세로 높이 속성

우선 주의할 점은, 단말기의 너비나 높이가 실제 브라우저의 너비나 높이와는 다르다는 것을 알아야 한다.
여하튼 사용하려면 device-width와 같이 앞에 device-만 붙여주면 되고, 위에서 사용한 것과 같이 쓰면 됨.

화면 회전 속성

스마트폰이나 태블릿에서는 화면을 회전할 수 있는데, 미디어 쿼리의 "orientation" 속성을 이용해 기기의 방향을 확인할 수 있다.
orientation의 속성값으로 portrait(기본 모드(세로)), landscape(가로)가 있다.
기본 문법: orientation: (portrait | landscape)

미디어 쿼리의 중단점

보통 화면의 크기를 기준으로 중단점(break point)를 설정한다.
일반적으로 모바일의 레이아웃을 기본으로 CSS를 만들고, 사양이 좀 더 좋고 화면이 큰 태블릿이나 데스크톱아 맞춰 더 많은 기능과 스타일을 추가하는데,
이렇게 모바일을 먼저 고려하여 미디어 쿼리를 작성하는 것을 "모바일 퍼스트" 기법이라고 한다.

일반적인 중단점

  • 스마트폰 : 미디어 쿼리를 사용하지 않고 기본 CSS로 작성. 만약 스마트폰의 회전 여부까지 신경쓴다면 세로와 가로모드를 각각 portrait 320px, landscape 480px로 지정.
  • 태블릿 : 세로 크기가 768px 이상이면 태블릿으로 지정, 가로 크기는 데스크톱과 동일하게 1024px 이상으로 지정
  • 데스크톱 : 화면 크기가 1024px 이상이면 데스크톱으로 설정

외부 CSS 파일 연결

link를 사용하는 방법과 @import를 써서 CSS를 연결하는 방법이 있는데, 주로 link를 사용(@import를 쓰면 자바스크립트보다 늦게 불러와서 많은 CSS파일을 연결 시 오류 발생 가능)

그래서 보통 일케 쓴다 -> (link rel="stylesheet" media="미디어 쿼리 조건" href="css 파일 경로")
여기서 미디어 쿼리 조건은 위에서 알아본 기본 문법과 거의 동일한데,
예를 들어 일케 하면 됨. media="only screen and (min-width: 321px) and(max-width:768px)"
