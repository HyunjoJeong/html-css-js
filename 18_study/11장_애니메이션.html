CSS Animation

css animation를 이용하면 자바스크립트를 이용하지 않고도 웹 요소에 애니메이션을 추가할 수 있다.
animation 속성은 특정 지점에서 스타일을 바꾸면서 애니메이션을 만드는데, 이런 스타일이 바뀌는 지점을 키프레임이라고 한다.
@keyframe을 이용해 키프레임을 정의한 후 animation 속성과 그 하위속성을 이용해 애니메이션의 실행시간이나 반복 여부를 지정한다.

종류설명
@keyframes 애니메이션이 바뀌는 지점을 지정
기본형:
@keyframes (이름) {
(선택자) { (스타일) }
}
animation-name @keyframes로 설정해 놓은 중간 상태를 지정
기본형: animation-name: (키프레임 이름) | none
animation-delay 지연 시간
animation-direction 애니메이션을 종류한 뒤 처음부터 시작할지, 역방향으로 진행할지 지정
기본형: animation-direction: normal | reverse | alternate | alternate-reverse
normal(default) : from --> to
reverse : to --> from
alternate : from --> to --> from --> to ..
alternate-reverse : to --> from --> to --> from ..
animation-duration 애니메이션의 실행 시간 지정
animation-iteration-count 애니메이션의 반복 횟수 지정
기본형: animation-iteration-count: 숫자 | infinite
animation-timing-function 키프레임의 전환 형태 (효과의 속도 곡선) 지정
기본형: animation-timing-function: (transition과 동일)
